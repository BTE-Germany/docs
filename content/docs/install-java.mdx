Die Installation von Java ist ziemlich einfach. Du musst die lediglich über folgenden Link den Installationsassistenten herunterladen: [https://adoptium.net/de/temurin/releases](https://adoptium.net/de/temurin/releases)

Die heruntergeladene Datei kannst du mit einem Doppelklick ausführen. Daraufhin sollte sich der Setup-Assistent öffnen. ![](https://d4rdaoypx775z.cloudfront.net/uploads/2026/02/chrome_HQWClzNQYd.png)


Normalerweise musst du keine der Einstellungen während der Installation verändern und solltest nur zweimal auf "_Weiter_" und dann auf "_Installieren_" klicken.

![](https://d4rdaoypx775z.cloudfront.net/uploads/2026/02/chrome_CGreRqlz8V.png)

Java sollte nun auf deinem System installiert sein und du solltest den Modpack Installer starten können.
